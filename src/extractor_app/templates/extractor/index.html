About This Project

Who

This was created for teachers and students as a means of Comprehension Testing and quizzing.

Why

As a final year project, this was a motivating topic which serves great educational purpose.

How

This project is implemented using Natural Language Processing, Machine Learning and Semantic Analysis to be able to extract questions from the inputted text.

Tech Stack

Python 3

Postgres

Amazon Web Services

Django Rest Framework

About the Creator

I, Traian Svinti, am a final year student in Dublin City University studing Computer Applications and Software Engineering. The scope of this project is to create a unique and useful idea which implements Natural Language Processing as it's main component.

When beginning this project, I wanted to use tools that I have not previously had experience with to improve my tech stack and knowledge of new systems.

I aim to maintain and improve this extractor as my Python skills improve.
